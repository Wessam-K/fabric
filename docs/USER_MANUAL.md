## 1. التثبيت / Installation

### تثبيت تطبيق سطح المكتب / Desktop App Installation

1. قم بتحميل ملف `WK-Factory Setup X.X.X.exe` / Download `WK-Factory Setup X.X.X.exe`
2. شغّل المُثبِّت واتبع التعليمات / Run the installer and follow the prompts
3. سيُنشئ التطبيق قاعدة بيانات تلقائياً عند أول تشغيل / The app creates the database automatically on first launch

### متطلبات النظام / System Requirements

| المتطلب / Requirement | الحد الأدنى / Minimum |
|---|---|
| نظام التشغيل / OS | Windows 10+ |
| الذاكرة / RAM | 4 GB |
| المساحة / Disk | 500 MB |

---

## 2. الإعداد الأولي / First-time Setup

عند أول تشغيل، ستظهر صفحة الإعداد:

1. **اسم المصنع**: أدخل اسم مصنعك
2. **المدير الرئيسي**: أنشئ حساب superadmin
   - اسم المستخدم (3 أحرف على الأقل)
   - كلمة مرور قوية (8 أحرف، حرف كبير، رقم، رمز خاص)
3. اضغط **إعداد النظام**

> ⚠️ احتفظ ببيانات الدخول في مكان آمن — لا يمكن استردادها بدون superadmin

---

## 3. تسجيل الدخول / Login

1. أدخل اسم المستخدم وكلمة المرور
2. إذا كان 2FA مُفعلاً: أدخل رمز المصادقة من تطبيق Authenticator
3. بعد 5 محاولات فاشلة، يُقفل الحساب مؤقتاً (15 دقيقة)

---

## 4. لوحة التحكم / Dashboard

لوحة التحكم تعرض:

- **بطاقات المؤشرات**: أوامر إنتاج نشطة، إيراد شهري، مخزون، فواتير معلقة
- **الرسوم البيانية**: أداء الإنتاج والإيرادات عبر الأشهر
- **التنبيهات**: مخزون منخفض، صيانة مستحقة، فواتير متأخرة
- **آخر الأوامر**: أحدث أوامر الإنتاج

اضغط على أي بطاقة للانتقال إلى الصفحة المتعلقة.

## 5. إدارة الأقمشة والإكسسوارات / Fabrics & Accessories

### إضافة قماش جديد
1. اذهب إلى **الأقمشة** من القائمة الجانبية
2. اضغط **قماش جديد**
3. أدخل: الكود، الاسم، النوع (أساسي/بطانة)، السعر لكل متر، المورد
4. اضغط **حفظ**

### استيراد جماعي
1. اضغط **استيراد CSV**
2. حمّل نموذج CSV
3. املأ البيانات
4. ارفع الملف

---

## 6. إدارة الموديلات / Models

### إنشاء موديل
1. اذهب إلى **الموديلات** → **موديل جديد**
2. أدخل كود الموديل والتفاصيل
3. أضف متطلبات الأقمشة والإكسسوارات (BOM)
4. حدد مراحل الإنتاج
5. اضغط **حفظ**

### قائمة المواد (BOM)
لكل موديل، حدد:
- الأقمشة المطلوبة مع الكمية لكل قطعة
- الإكسسوارات المطلوبة مع الكمية لكل قطعة

---

## 7. أوامر الإنتاج / Work Orders

### إنشاء أمر إنتاج
1. اذهب إلى **أوامر الإنتاج** → **أمر جديد**
2. اختر الموديل
3. أدخل الكمية وتوزيع المقاسات
4. حدد تاريخ التسليم وخط الإنتاج
5. اضغط **حفظ**

### متابعة المراحل
كل أمر إنتاج يمر بمراحل:
```
مسودة → قص → قيد التنفيذ → مكتمل → تم التسليم
```

### تسجيل تقدم المرحلة
1. افتح أمر الإنتاج
2. اذهب لتبويب **المراحل**
3. أدخل الكمية المنجزة
4. اضغط **تقدم المرحلة**

---

## 8. الفواتير والمحاسبة / Invoices & Accounting

### إنشاء فاتورة
1. اذهب إلى **الفواتير** → **فاتورة جديدة**
2. اختر العميل
3. أضف بنود الفاتورة (الكمية × السعر)
4. حدد شروط الدفع والخصم
5. اضغط **حفظ**

### تسجيل دفعة
1. افتح الفاتورة
2. اضغط **تسجيل دفعة**
3. أدخل المبلغ وطريقة الدفع

### دليل الحسابات
- أصول، خصوم، حقوق ملكية، إيرادات، مصروفات
- قيود يومية مع تساوي المدين والدائن

---

## 9. الموارد البشرية / HR

### إدارة الموظفين
- إضافة موظف: الاسم، الوظيفة، القسم، الراتب
- تسجيل الحضور اليومي
- إدارة الإجازات

### كشوف الرواتب
1. اختر الشهر
2. تحقق من بيانات الحضور
3. اضغط **معالجة الرواتب**
4. راجع الحسابات
5. اعتمد واطبع القسائم

---

## 10. النسخ الاحتياطي / Backup & Restore

### إنشاء نسخة احتياطية
1. اذهب إلى **النسخ الاحتياطي**
2. اضغط **نسخ احتياطي الآن**
3. حمّل الملف واحتفظ به في مكان آمن

### استعادة نسخة احتياطية
1. اذهب إلى **النسخ الاحتياطي**
2. اضغط **استعادة**
3. اختر ملف النسخة
4. أكّد العملية

> ⚠️ الاستعادة تستبدل جميع البيانات الحالية

### النسخ التلقائي
- يمكن تفعيل النسخ التلقائي من **الإعدادات**
- الفاصل الزمني الافتراضي: 24 ساعة

---

## 11. الأمان / Security

### المصادقة الثنائية (2FA)
1. اذهب إلى **الملف الشخصي**
2. اضغط **تفعيل المصادقة الثنائية**
3. امسح رمز QR بتطبيق Authenticator
4. أدخل رمز التحقق
5. احتفظ بأكواد الطوارئ

### إدارة الجلسات
- عرض الجلسات النشطة من **الملف الشخصي**
- إنهاء أي جلسة مشبوهة

### الصلاحيات والأدوار
| الدور | الوصول |
|---|---|
| superadmin | وصول كامل |
| manager | جميع العمليات (ما عدا الإعدادات) |
| accountant | المالية والتقارير |
| production | أوامر الإنتاج والجودة |
| hr | الموارد البشرية |
| viewer | عرض فقط |

## 12. استكشاف الأخطاء / Troubleshooting

| المشكلة / Problem | الحل / Solution |
|---|---|
| لا يمكن تسجيل الدخول | تحقق من اسم المستخدم وكلمة المرور. بعد 5 محاولات فاشلة، انتظر 15 دقيقة |
| الحساب مقفل | انتظر 15 دقيقة أو اطلب من المدير إلغاء القفل |
| الصفحة لا تحمّل | أعد تحميل الصفحة (F5 أو Ctrl+R) |
| خطأ في الحفظ | تحقق من اتصال الإنترنت واملأ جميع الحقول المطلوبة |
| بيانات مفقودة | استعد نسخة احتياطية حديثة |
| التطبيق بطيء | أغلق التبويبات غير الضرورية وأعد تشغيل التطبيق |
